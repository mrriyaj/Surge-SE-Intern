## Run Locally (Recommended)

Clone the project

```bash
  git clone https://github.com/riyajkafar/Surge-SE-Intern.git
```

#Run Backend

Go to the project directory

```bash
  cd Backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

#Run Frontend

Go to the project directory

```bash
  cd Frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
